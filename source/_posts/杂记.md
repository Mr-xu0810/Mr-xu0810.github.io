# 杂记

## windows telnet设置
#### 开启

在控制面板->程序与功能->启动或关闭windows功能->找到Telnet Client勾选并点击确定，等待完成即可

#### 测试

```bash
  telnet 10.32.10.2   # ip测试
  telnet 10.32.10.2 8080 # 端口测试
```
